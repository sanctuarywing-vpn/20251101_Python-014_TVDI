## 📖 基本操作

### 啟動後

```
1. 應用自動載入行業清單 (等待「就緒」狀態)
2. 在「🏭 行業」選擇行業 (如: 半導體)
3. 在「🔍 搜尋」輸入股票代碼或名稱
4. 雙擊股票或點擊「➕ 加入」添加到觀察
```

### 查看卡片

```
1. 股票自動顯示在右側卡片區
2. 窗口變窄時卡片自動重排
3. 點擊卡片右上角「✕」移除股票
```

### 查看熱圖

```
1. 左下方自動顯示市場熱圖
2. 綠色 = 上漲, 紅色 = 下跌
3. 每次更新時自動更新熱圖
```

### 自動更新

```
1. 勾選「自動更新 (每分鐘)」
2. 每 60 秒自動更新一次
3. 時間顯示在工具列右側
```

## ❓ 常見問題

### Q: 啟動時卡住？

**A**: 
- 確認 `taiwan_stock_crawler.py` 存在
- 檢查 Python 版本 (3.7+ 推薦)
- 查看控制台錯誤信息

### Q: 卡片顯示不全？

**A**: 
- 放大窗口寬度
- 或減小 `card_width` 值
- 卡片會自動換行

### Q: 如何調整視窗大小？

**A**: 
- 拖拽視窗邊界即可
- 卡片會自動重新排列

### Q: 美股什麼時候推出？

**A**: 預計 1-2 週內實現

### Q: 如何回到標準版？

**A**: 
```bash
python stock_monitor_gui.py
```

### Q: 卡片太多很慢？

**A**: 
- 建議不超過 100 支股票
- 觀察清單不要過多

## 🔧 故障排查

### 錯誤 1: ImportError

```
Error: No module named 'taiwan_stock_crawler'
```

**解決**: 確認在 lession8_1 目錄執行

```bash
cd lession8_1
python run_v2.py
```

### 錯誤 2: 卡片不更新

```
卡片資訊沒有變化
```

**解決**: 
1. 點擊「🔄 立即更新」
2. 或勾選「自動更新」

### 錯誤 3: 熱圖為空

```
市場熱圖不顯示
```

**解決**: 
1. 等待應用完全載入
2. 手動點擊「立即更新」

### 錯誤 4: 滾輪不工作

```
無法用滾輪滾動卡片
```

**解決**: 
- 先點擊卡片區域
- 再使用滾輪
